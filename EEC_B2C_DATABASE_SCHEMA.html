Version: 1.0

Database Type: MongoDB (Mongoose Models)

1. Overview

This schema is generated from the current backend model definitions in backend/src/models. The platform uses MongoDB collections mapped by Mongoose schemas with timestamps, references, enums, and selected indexes.

5. Notes

  • Most collections enable timestamps: true, adding createdAt and updatedAt.
  • MongoDB collection naming is auto-pluralized by Mongoose unless overridden.
  • Question schema is polymorphic and stores multiple question subtypes in one collection.
