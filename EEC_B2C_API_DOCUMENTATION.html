2. Authentication & Authorization

For protected routes, send:

Authorization: Bearer <JWT_TOKEN>
Access LabelMeaning
PublicNo login required
AuthAny authenticated user
AdminAdmin-only by explicit route check
TeacherTeacher-only by explicit route check
Admin/TeacherEither admin or teacher
